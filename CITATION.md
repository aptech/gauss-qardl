# Citation

If you use the QARDL GAUSS library in academic work, reports, teaching
materials, software comparisons, or applied research, please cite the software
and the underlying econometric methodology.

## Software Citation

```bibtex
@software{clower_qardl_gauss_2026,
  author       = {Clower, Eric},
  title        = {{QARDL for GAUSS: Quantile Autoregressive Distributed Lag Estimation, Inference, and Applied Workflows}},
  year         = {2026},
  version      = {3.0.0},
  publisher    = {Aptech Systems, Inc.},
  url          = {https://github.com/aptech/gauss-qardl},
  note         = {GAUSS application package}
}
```

The software description article is available at:

```text
docs/QARDL_RELEASE_ARTICLE.md
docs/QARDL_RELEASE_ARTICLE.pdf
```

## Methodology Citation

```bibtex
@article{cho_kim_shin_2015_qardl,
  author  = {Cho, Jin Seo and Kim, Tae-Hwan and Shin, Yongcheol},
  title   = {Quantile cointegration in the autoregressive distributed-lag modeling framework},
  journal = {Journal of Econometrics},
  year    = {2015},
  volume  = {188},
  number  = {1},
  pages   = {281--300},
  doi     = {10.1016/j.jeconom.2015.05.003}
}
```

When using ARDL bounds testing, robust/HAC covariance, bootstrap intervals, or
quantile regression theory directly, also cite the relevant methodological
sources listed in the README and usage guide.
